# Application Transfer Checklist

## For Sellers

### Pre-Sale Preparation
- [ ] Document all custom configurations
- [ ] Export database schema and sample data
- [ ] Create comprehensive deployment guide
- [ ] Test deployment on fresh accounts
- [ ] Document any customizations made

### During Transfer
- [ ] Provide access to source code repository
- [ ] Share deployment documentation
- [ ] Export database (schema + data if included in sale)
- [ ] Provide list of all environment variables needed
- [ ] Transfer any custom domain configurations

### Post-Sale
- [ ] Buyer confirms successful deployment
- [ ] Remove buyer's access to your accounts
- [ ] Transfer GitHub repository ownership (if applicable)

## For Buyers

### Setup Verification
- [ ] Create new Supabase project
- [ ] Create new Groq account  
- [ ] Deploy application to Vercel
- [ ] Configure all environment variables
- [ ] Run database setup scripts
- [ ] Test all major features:
  - [ ] User registration/login
  - [ ] Company registration flow
  - [ ] Blog management
  - [ ] Q&A forum
  - [ ] AI assistant
  - [ ] Admin dashboard

### Customization
- [ ] Update branding and styling
- [ ] Configure custom domain
- [ ] Set up monitoring and analytics
- [ ] Configure backup procedures
